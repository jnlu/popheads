Useful rate machine toggles

  • HOST | Add this line under the "Username" line for a host's set of scores. The first set of scores must be from a host. You can designate multiple people as a host. See any Sample for guidance.
  • NO ALBUM | Add this line under the first "HOST" line if your rate is a collection of songs with no connection (i.e. not a series of albums or not from a small group of artists). See Sample 4.
  • CATEGORY: | Add this line to designate that this rate is categorized into something other than albums. For example, type "CATEGORY:Artist" to designate this rate as a collection of artists. Whatever you type after the colon replaces the "Album:" line; e.g. it becomes "Artist:" instead. See Sample 5.
  • ARTIST: | Add this line after a category line to link every song in that category to that artist. The artist name gets printed alongside the song name in the results section. See Samples 2 and 5.
  • PRINTALLSCORES | Add this line under a "HOST" line to print every individual score in the detailed results section instead of just the highest and lowest scores. This feature is recommended unless you have an extremely high number of participants.

See the rate server for information on other toggles.
